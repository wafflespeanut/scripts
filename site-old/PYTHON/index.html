Basic data structures: lists & sets...



							string = '''Hello, world!'''
							list_ = list(string)		# print list_
							set_ = set(string)			# print set_

							set(list(string)) == set(string)	# True
							set(list(string)) is set(string)	# False?

							[] == [] and [] is []		# False?
							256 is 32 * 8				# (small integers) True
							65536 is 256 * 256			# (big ones) False?

							"zzz" is "z" * 3
							x = "z"				# a binding changes the result
							"zzz" is x * 3		# Meh...

Basic data structures: Immutable tuples & hashed dicts...



							tuple_ = tuple(range(10))		# immutable
							tuple_[0] = 1				# wrong!

							dict_ = zip(range(1, 10), range(1, 10))
							dict_ = zip(enumerate(range(1, 10)))

							# awesomeness of dictionaries...
							dict_[0] = 'Meow...'
							dict_['dog'] = 'Woof!'			# apparently, we can do anything!
							dict_[tuple(range(3))] = True
							dict_[range(3)] = False			# well, almost...

							print "{} {}".format(dict_[0], dict_['dog'])
